Healthy Appetite

Objectives

Design a responsive web app for healthy eating habits, that users can access easily on any device and any places. It helps users to change their lifestyle and stay on track with it. Providing support for a healthier lifestyle.

Challenge

Users can search for recipes by ingredients and save their favorite meals. To assist on the lifestyle change users can create a weekly meal plan and get help with the shopping list.

My approach

I started with competitive analyses and user research that helped me to create the user personas for my project. After defining the user flows, I sketched up the first wireframes and tested the prototype. Finishing this part of the project, I started to plan and design the UI for the app.
