Healthy Appetite

Objectives

Design a responsive web app for healthy recipes that users can access easily anywhere on any device. It supports users who want to change their lifestyle and stay on track with it. It has to provide instructions that are easy to follow for anyone with a different cooking experience.

Challenge

Users can search for recipes by ingredients and save their favourite meals. They receive recommendations based on the personalisation step that filled in when signing up for the app. To assist with the lifestyle change, users can create a weekly meal plan and get help with the shopping list.

My approach

I began with competitive analyses and user research. It helped me to create the user personas for my project. After defining the user flows, I sketched up the first wireframes and tested the prototype. After finishing this part of the project, I started to plan and build the UI of the app.
